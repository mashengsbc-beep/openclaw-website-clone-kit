# Rebuild Spec Template

Use this exact outline when producing a website rebuild brief.

## Target
- URL:
- Date inspected:
- Goal:
- Fidelity target:

## Summary
- What the page is
- What makes it distinctive
- Whether it is a good clone candidate

## Section Inventory
List sections from top to bottom.
For each section include:
- working name
- purpose
- notable content
- notable layout/style
- interaction hints

## Style Notes
- typography
- palette
- spacing rhythm
- border radius / shadows
- imagery style
- motion hints

## Assets
- logos
- hero images
- icons
- videos
- downloadable media to localize

## Proposed Implementation
- suggested stack
- routes/pages needed
- component list
- content/data shape
- folder structure

## Risks / Gaps
- what was not verified
- what likely needs manual polish
- what may be legally sensitive

## Recommended Next Step
Choose one:
- stop at analysis
- scaffold project
- build first-pass homepage
- do deeper browser inspection
